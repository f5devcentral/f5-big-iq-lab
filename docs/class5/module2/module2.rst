Module 2: BIG-IP device cluster status and configurations
=========================================================

BIG-IQ provides visibility in to the BIG-IP cluster configuration and status. You can review the sync status, drill down to review cluster configuration, and initiate cluster sync activities from BIG-IQ.

**BIG-IP Clusters - (DSC) Device Service Clusters**

Device Service Clustering, or DSC, is a BIG-IP TMOS feature that lets you organize BIG-IP devices in groups allowing for synchronization of configuration objects.
